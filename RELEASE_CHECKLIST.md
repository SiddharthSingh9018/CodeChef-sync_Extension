# Release Checklist

## Before A Release

- Reload the unpacked extension in Chrome
- Test CodeChef problem-page sync
- Test CodeChef `viewsolution` sync
- Test Codeforces problem-page sync
- Test LeetCode visible-page sync
- Test NeetCode visible-page sync
- Confirm README instructions still match the current behavior
- Confirm the changelog has a new entry

## GitHub Polish

- Update the repo description
- Update repo topics
- Add screenshots or a demo GIF
- Tag a release
- Upload a zip of the extension source if needed

## Suggested Repo Topics

- `chrome-extension`
- `competitive-programming`
- `leetcode`
